To install, simply run the following command, then click on PandaBot.py
python setup.py develop

If you wish to disable a command, do the following:
1. Open the rolebot.py file
2. Go to the line starting with "linked_commands"
3. Find the long and short names of the command you want
4. Go to the line starting with "commands", find the long command in the list, and change its "True" to "False"
5. Go to the line starting with "short_commands", find the short command in the list, and change its "True" to "False"

!!!DO NOT EDIT CONFIG.TXT, LOG.TXT, or ROLES.TXT DIRECTLY!!!
